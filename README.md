# Fiber Optic Weigh In Motion Sensor
  This project is done to develop optical fiber sensor to measure weigh of moving vehicle.
 <img src="https://github.com/Schildsladder/Fiber-Optic-Weigh-Sensor/blob/master/pictures/a.jpg" width="50%">
## Usage
1. Install required modules with `pip install -r requirements.txt`.
2. Run the program with `python mandelbrot_generator.py`
3. Wait for the program to compute
4. Check the notebook file
